SmartTax Agent - User Guide

📌 Overview

The SmartTax Agent is an intelligent, conversational tax assistant designed to guide you step-by-step through tax calculations, helping you compare multiple scenarios with detailed IRS-referenced explanations. Whether you're optimizing deductions, exploring different filing strategies, or planning ahead, this AI-powered tool makes tax filing easier and more insightful—all for free.

🏆 User Roles vs Key Features

The SmartTax Agent can be utilized by either Individual Users for preparing their taxes for maximum refund or Tax Specialists for preparing and reviewing their Clients tax returns.

🎯 How It Works

1️⃣ Enter Tax Details Manually
  • Basic info: Filing status, age, dependents
  • Income: W-2, 1099, investments, self-employment, etc.
  • Deductions: Mortgage payments, IRA contributions, tuition fees (1098-T), and more
2️⃣ Validate Document/Information Summary

Before proceeding with calculations, review a summary of the details entered.

3️⃣ Tax Calculation Mapped to IRS Forms
  • Step-by-step breakdown of how each number is derived
  • References to IRS forms like 1040, Schedule A, Schedule C, etc.
  • Clear explanation of why specific deductions apply (or don’t)
4️⃣ Play Out Different Scenarios with Side-by-Side Comparisons

Want to try different scenarios and see how they impact your refund?

  • Traditional IRA vs. Roth IRA – from $0 to full match
  • What if parent or student pays tuition fees (1098-T)?

For any scenario, ask "How can I get a full refund?"
